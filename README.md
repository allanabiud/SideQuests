# SideQuests

A task management app for real-life adventures.

---

## Getting Started

### Prerequisites

- Flutter SDK ^3.11.0
- Android Studio, Neovim or VS Code with Flutter extensions
- Android SDK 29+

### Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/allanabiud/SideQuests.git
   ```

2. **Setup Dependencies**

   ```sh
   flutter pub get
   ```

3. **Code Generation**

   Since this project uses code generation for Riverpod and AutoRoute, run:

   ```sh
   dart run build_runner build --delete-conflicting-outputs
   ```

4. **Run the project**
   - Connect a device or start an emulator
   - Click Run or use `flutter run`
